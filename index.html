How It Works

QuickShut is designed to be intuitive and easy to use. Here's how to get started:

1

Choose Mode

Select between Date/Time mode or Custom Countdown mode using the toggle switch.

2

Select Action

Pick what you want to happen: Shutdown, Restart, Log Off, or Lock.

3

Set Time

Enter your desired shutdown/restart/log off/lock date and time or set a custom countdown duration.

4

Start Timer

Click the "Start" button to begin the countdown to the action.

5

Monitor or Modify

Watch the countdown, pause if needed, or reset to start over.

Frequently Asked Questions

Is QuickShut free to use?

Yes, QuickShut is completely free to download and use with no hidden fees or premium features.

Which operating systems are supported?

QuickShut is currently designed for Windows operating systems. Support for macOS and Linux may be added in future updates.

Can I schedule recurring actions?

The current version doesn't support recurring schedules, but this feature is planned for a future update.

Will QuickShut save my unsaved work before shutdown?

QuickShut uses Windows' standard shutdown procedure, which will prompt you to save unsaved work in most applications. However, we recommend saving your work regularly when a shutdown or restart is scheduled.

Can I also schedule restarts or locking my PC?

Yes! In version 1.3, you can now schedule Shutdown, Restart, Log Off, and Lock actions directly from QuickShut.
